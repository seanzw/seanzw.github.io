Projects

Pedestrain Tracker

This is a c++ system using online-boosting classifier and particle filter to track and count pedestrains in a video clip. The video is uncalibrated (without depth information).

Project Repo: https://github.com/seanzw/PedestrainTracker

Yart-cpp

Yet another ray tracer in c++. It supports direct light and bidirectional path tracing. For BSDF, it supports perfect specular, perfect refraction and Cook-Torrance.

Project Repo: https://github.com/seanzw/yart-cpp

MicroPython on FPGA

MicroPython is implementation of Python for embedded system. Under supervision of Peter Y K Cheung at Imperial College London, Zhixun Tan and I port MicroPython to Altera's DE0-Nano-SoC. We also write a simple Python module to communicate with the FPGA part.

Project Repo: https://github.com/seanzw/MicroPythonFPGA
